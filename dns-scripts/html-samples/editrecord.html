Edit DNS Record

Record Type. For a reverse DNS zone, the only allowed type is "PTR". For a "normal" (forward) zone, it may be one of:

Record name is a DNS name such as "www" or "ftp.albocha.de".

Record Name:

Record Value depends on the record type. For "A" it is an IPv4 address, for "AAAA" it is an IPv6 address. For "CNAME" it is a DNS name. For "NS" and "MX" it can bei an IPv4 or IPv6 address or a DNS name. For "TXT" and "SPF" it is an arbitrary string. "SRV" is a bit special.

Record Value:

The MX prio text input field should only be displayed if the "Record type" is MX. Value must be a positive integer.

MX priority:

The TTL text input field should be greyed out unless "Custom TTL" is chosen. Value must be a positive integer (seconds). Might also be a select with discrete values like "5min", "1h".
